# Canonical document format

Future normalization produces Markdown without changing the original source. It must preserve
heading hierarchy, ordered content, tables where representable, links, and relative media
references. Source markers connect extracted PDF pages to source locations. Known losses,
unsupported content, and uncertain ordering are recorded in conversion warnings. Markdown, DOCX,
and text-based PDF normalization are implemented; OCR and complete embedded-media extraction remain
unsupported.
